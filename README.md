# Joy Ram Das — Portfolio

A modern, full-stack developer portfolio built with Next.js 15, Framer Motion, and Tailwind CSS.

## Tech Stack
- **Next.js 15** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion** — animations & transitions
- **Lucide React** — icons

## Features
- 🎯 Custom animated cursor with follower
- 💻 Terminal-style UI components throughout
- ⚡ Animated SVG code orbit on hero
- 🔤 TypeWriter effect for roles
- 📊 Animated skill bars on scroll
- 🌊 Smooth section reveal animations
- 📱 Fully responsive
- 🎨 Noise texture + grid background
- 🌀 Scanline effect
- 🎨 Gradient borders with hover effects

## Setup

```bash
# 1. Install dependencies
npm install

# 2. Run development server
npm run dev

# 3. Open in browser
http://localhost:3000
```

## Deployment
```bash
# Deploy to Vercel
npx vercel

# or build for production
npm run build
npm start
```

## Customization

All personal data is in the component files:
- `components/sections/Hero.tsx` — name, title, social links, stats
- `components/sections/About.tsx` — bio, fun facts
- `components/sections/Skills.tsx` — skill categories and levels
- `components/sections/Experience.tsx` — work history
- `components/sections/Projects.tsx` — projects
- `components/sections/Contact.tsx` — contact info
